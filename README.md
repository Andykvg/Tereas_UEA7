# Códigos UEA 7
# Andrea K. Villegas García ***Producción y comunicación colaborativa*** _Artes y Comunicación Digitales_

## *Objetivo* 
El objetivo de este repositorio es el de almacenar los distintos ejercicios y tareas que se realicen durante las clases del eje tecnológico de la UEA 7 "Producción y comunicación colaborativa". Se pretende organizar los trabajos con el fin no solo de utilizarlos durante el transcurso de este trimestre, sino también para utilizar el repositorio como recurso de apoyo en futuros proyectos. Además, y como objetivo puntual, el repositorio contendrá la descripción de los proyectos, la cual agrega un nivel de organización que, al momento de ser recuperados los códigos, se podrá consultar con detenimiento cada una de sus funciones.
